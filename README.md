# Hybrid Financial Recommender

**Тема проекта:** реализация гибридной рекомендательной системы для онлайн-сервиса финансовых услуг.  
Проект объединяет контентный и коллаборативный подходы для персонализации предложений по кредитам, депозитам и страховкам.

## Что делает система

1. Проверяет наличие данных — если их нет, генерирует синту.
2. Обрабатывает и нормализует данные о продуктах.
3. Обучает две модели:
   - **Content-based** (по признакам продуктов)
   - **Collaborative** (по пользовательским рейтингам)
4. Комбинирует результаты в **гибридную модель**.
5. Через интерфейс **Gradio** позволяет ввести `user_id` и получить персональные рекомендации.

## Для теста

```bash
pip install -r requirements.txt
python app.gradio_app
```

Откроется Gradio UI:
Введите `User ID` (1–999) -> получите финансовые рекомендации.

## Дальнейшие шаги

- **Ввести метрики качества** (Precision@K, NDCG, MAP).
- **Добавить факторизацию или Embedding-based подходы** (например, с LightFM или PyTorch).
- **Интеграция с CRM** — подключить API реального онлайн-сервиса.
- **A/B-тестирование** — сравнение гибридной модели с классическими методами.
- **Построить RAG для рекомендаций**: текстовые объяснения “почему система предложила этот продукт”.
- **Внедрить LLM**: генерация рекомендательного сообщения для простого восприятия.
